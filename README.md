# RedSeaOS

### RedSeaOS – Independent Linux Distribution by a 13-Year-Old Developer

> **Project Status Notice**
>
> The RedSeaOS GitHub repository was recently created and currently contains only the README and LICENSE files. The previous ShiftOS repository has been removed as the project evolved into RedSeaOS.
>
> RedSeaOS is currently in the early stages of development. Planned components include the **Apex Desktop Environment**, **Shift Package Manager**, **custom icon theme**, **custom fonts**, **unique visual branding**, **OpenRC init system**, and an **optimized Linux 6.18 LTS kernel**. More code, documentation, and project files will be added as development progresses.

RedSeaOS is an independent Linux distribution developed by Aikid7, created with a focus on simplicity, performance, reliability, and complete system integration. The goal of the project is to build a complete Linux ecosystem with its own desktop environment, package manager, branding, and user experience.

## ✨ Planned Features

- Apex Desktop Environment
- Shift Package Manager
- Custom icon theme
- Custom fonts
- Unique visual branding
- OpenRC init system
- Optimized Linux 6.18 LTS kernel
- Lightweight and performance-focused design
- Open-source development model
